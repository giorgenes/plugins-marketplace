# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this is

The `understand` skill for the `sw-eng` plugin in the [plugins-marketplace](../../../../). It gives Claude a structured workflow for deeply understanding an issue (like from Linear, Github, Jira, etc).

## Structure

Single file: `SKILL.md` — contains the skill instructions loaded by the marketplace when the skill is invoked.
